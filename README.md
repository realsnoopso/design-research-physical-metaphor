# 실제 상황을 모티브로 한 인터랙션 디자인 레퍼런스

Physical metaphor in UI/UX design - 현실 세계의 물리적 동작을 디지털 인터랙션으로 변환한 사례와 이론 모음

## 🎯 핵심 개념

좋은 인터랙션 디자인은 **익숙한 메타포와 패턴**을 반복해서 사용합니다. 사용자는 배운 것을 계속 응용하면서 점점 더 잘 다루게 됩니다.

현실 세계의 물리적 동작을 디지털 인터랙션으로 변환:
- 📖 **책 넘기기** → 스와이프
- 🃏 **카드 밀기** → 앱 닫기  
- 🤏 **손가락 집기** → 핀치 줌
- 💨 **물리적 속도** → 애니메이션 속도

단, 디지털 환경에 맞게 조정됩니다 (예: 핀치 방향 반전).

## 💡 핵심 인사이트

### Dynamic Island = 물리 법칙
- 사용자가 던진 속도와 각도 그대로 유지
- 중앙에서 완벽히 멈추지 않음
- 현실의 카드 밀기와 비슷

### Spring > Easing
- Spring은 자연스럽게 느껴짐 (예측 가능)
- 객체가 움직이고 튀는 방식처럼
- 제스처 중단/재타겟팅 처리 가능

### Haptic으로 메타포 강화
- 무거운 물체 떨어뜨리기 시뮬레이션
- 시각 + 햅틱 (Heavy 충격) + 소리 (공명)
- 3가지 감각 조합

### 핀치 방향 반전의 이유
- 현실: 손가락 벌림 → 오므리기 (집기)
- 디지털: 손가락 붙임 → 벌리기 (확대)
- **이유:** 확대 위치를 정확하게 짚기 위해
